Clover v2 to bootloader typu open source bazujący na różnych projektach: Clover, rEFIt, XNU, VirtualBox. Podstawą Clover v2 jest ostatnie wydanie EDK2.

http://sourceforge.net/projects/cloverefiboot/

Główne funkcje:

  • Uruchamianie macOS 10.xx, Windows EFI oraz Linux EFI
  • Obsługa RuntimeServices dla systemów operacyjnych, nie dostępna w innych bootloaderach
  • Automatyczna detekcja sprzętu z możliwością ręcznej edycji ustawień w pliku. Konfiguracyjnym config.plist
  • Dzięki Clover można przełączać się między systemami za pośrednictwem panelu preferencji
  • UUID zapisany w config.plist będzie taki sam jak w OS.
  • SMBIOS poprawiany do standardu 2.6
  • Tabele ACPI poprawiane do standardu 4.0.
  • Niestandardowe tabele DSDT ładowane z bootowanej partycji lub z katalogu EFI
  • DSDT patching on the fly by fixes and by patterns;
  • Konfigurowalny RestartFix, poprawka sleep/wake oraz wstrzykiwanie PCIRootUID
  • Obsługa szerokiego zakresu kart graficznych ATI, Nvidia oraz Intel z możliwością ręcznej. Konfiguracji w pliku config.plist.
  • Poprawka dla USB (łącznie z obsługą USB 3.0)
  • Wstrzykiwanie ID dla karty dźwiękowej oraz informacji karty sieciowej
  • Generowanie stanów turbo, P-State oraz C-State w celu uzyskania natywnego zarządzania
  • Ładowanie tabel ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)
  • Extra kernel extensions loading;
  • Kernel and kext patching on the fly;
  • Set NVRAM variables for different purpose;
  • Domyślny rozruch systemu po upływie ustalonego czasu, konfigurowalne GUI oraz zapisywanie screenów z GUI za pomocą klawisza F10
  • Zapisywanie boot.log w OS oraz preboot.log w GUI za pomocą klawisza F2
