Clover v2 é de código aberto e baseado em diferentes projetos: Clover, rEFIt, XNU, VirtualBox. O EDK2 principal em última versão.

http://sourceforge.net/projects/cloverefiboot/

Principais Características:

  • Inicia macOS 10.xx, Windows EFI e Linux EFI
  • É um gerenciador de boot EFI que oferece RuntimeServices para o sistema operacional que é impossível com os bootloaders antigos
  • Inicialmente Clover faz a deteção automática de hardware e configura todas as opções padrão. Posteriormente o utilizador pode alterar no arquivo config.plist.
  • Com Clover você pode iniciar outro sistema operativo suportado através de Disco de Inicialização em Preferências do Sistema
  • UUID definido no config.plist será o mesmo no sistema
  • SMBIOS será corrigido para o padrão 2.6
  • ACPI será corrigido para o padrão 4.0.
  • Custom DSDT será carregado da partição iniciada ou da pasta EFI
  • DSDT patching na memoria;
  • RestartFix, sleep/wake e PCIRootUID personalizáveis
  • Propiedades de injeção de vídeo para ATI, NVidia e Intel com grande gama de placas suportadas. Também personalizável.
  • USB fix (LegacyOff, Ownership, Builtin, clock-id) até USB 3.0
  • Injeção de HDA e Ethernet integrado
  • Generador CPU turbo, P- e C-states
  • Carregamento de tabelas ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)
  • Extra kernel extensions (Kext) iniciando;
  • Kernel e kext patching na memória;
  • Configura NVRAM variaveis para objectivos diferentes;
  • Inicialização automática após contagem regressiva, Interface personalizável e faz capturas de tela do GUI pressionando a tecla F10
  • Salva boot.log no sistema e preboot.log pelo GUI apertando a tecla F2
