Clover v2.5k adalah proyek Open Source berdasarkan: Clover, rEFIt, XNU dan VirtualBox. Dibangun dengan EDK2 revisi terbaru.

http://sourceforge.net/projects/cloverefiboot/

Fitur-fitur Utama:

  • Boot macOS 10.xx, Windows dan Linux (EFI)
  • Clover menyediakan RuntimeServices EFI untuk OS yang mustahil bagi Bootloder Legacy
  • Clover mendeteksi Perangkat Keras secara otomatis dan menyetel Properties. Namun pengguna dapat mengubahnya lewat config.plist
  • Dengan Clover, Anda dapat Restart menuju OS lain dari preferensi Startup Disk
  • UUID yang tertera di config.plist akan sama dengan milik OS
  • SMBIOS akan dikoreksi ke Standard 2.6
  • ACPI akan dikoreksi ke Standard 4.0.
  • DSDT Custom dimuat dari partisi yang diboot / folder EFI
  • Patch DSDT On The Fly melalui fixes & patterns
  • RestartFix dan membenahi Sleep/Wake, serta nilai PCIRootUID
  • Injektor Properties ATI, NVidia dan Intel untuk berbagai jenis Kartu Grafis yang didukung. Bisa diatur kemudian.
  • Pembenahan USB (LegacyOff, Ownership, Built-in, clock-id) hingga USB 3.0
  • Injeksi HDA (High Definition Audio) dan Ethernet Built-in
  • Generator CPU Turbo serta P-States & C-States
  • Memuat Tabel ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)
  • Memuat kexts ekstra (fihak ketiga)
  • Patch Kernel dan Kext secara On The Fly
  • Menyetel variabel NVRAM untuk berbagai tujuan
  • Boot setelah Timeout, GUI yang bisa diubah, dan simpan screenshots GUI dengan tombol F10
  • Simpan boot.log di OS dan preboot.log dari GUI dengan tombol F2
