Clover v2 este un software open-source bazat pe diferite proiecte: Clover, rEFIt, XNU, VirtualBox. Principalul este EDK2 ultima revizie.

http://sourceforge.net/projects/cloverefiboot/

Facilitati principale:

  • boot pentru: macOS 10.xx, Windows EFI si Linux EFI
  • Bootloader-ul sau EFI ofera RuntimeServices pentru sistemul de operare, lucru imposibil pentru bootloader-ele traditionale de tip Legacy
  • in general, Clover ofera detectie hardware automata si setarea implicita a tuturor proprietatilor. Uilizatorul le poate insa modifica din config.plist
  • cu Clover, puteti alege sa initiati alt sistem de operare, folosind Startup Disk prefPane
  • UUID-ul folosit in config.plist va fi acelasi si in sistemul de operare
  • SMBIOS va fi corectat pentru standardul 2.6
  • ACPI va fi corectat pentru standardul 4.0.
  • Un DSDT modificat va fi incarcat de pe partitia de boot sau din folderul EFI
  • Corectie DSDT on-the-fly folosind fix-uri si modele predefinite
  • restartFix configurabil, fix-uri pentru sleep/wake si valoare PCIRootUID
  • Injector de Proprietati Grafice pentru ATI, NVidia si Intel pentru o arie larga de placi video suportate. De asemenea customizabile.
  • Solutii USB (LegacyOff, Ownership, Builtin, clock-id) pana la USB 3.0
  • Injector HDA si injector predefinit pentru Ethernet
  • CPU turbo si generator de stadii P si C
  • Incarcarea tabelelor ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)
  • incarcare kext-uri aditionale
  • corectii kernel si kext on-the-fly
  • seteaza variabile in NVRAM pentru diverse scopuri
  • initiere implicita dupa timeout, interfata configurabila, si salvarea de screenshot-uri din interfata, apasand F10
  • salvarea boot.log si preboot.log in sistemul de operare, din interfata grafica, apasand F2
